# Puppet Edge Example Repository

This repository contains a collection of example configurations and tasks for use with **Puppet Edge**. These examples are intended as starting points for your own implementation of network device management using Puppet Edge.

**Key points:**

- Some examples are generated using Puppet Infra Assistant and our Puppet MCP server. You should validate all examples with your specific systems before using them in production environments.
- These examples are provided for inspiration and to help you get started managing your networking devices with Puppet Edge.
- This repository is not supported software. Use these examples at your own risk and adapt them as needed for your environment.

See the included directories for specific device and task examples.
